QC Dance Wordpress Site

This design is built on WordPress using the Divi theme. I designed each page myself and used CSS and HTML when needed for customizations not available through the theme options. This 24 page website took a lot of planning and customizations due to the need for registration pages, class schedules, and detailed booking questionnaire's. The site you see is hosted in a staging environment since I no longer manage their production environment. Each image is scrollable and each heading above the displayed gallery pages is linked to be able to show the page's full functionality.
